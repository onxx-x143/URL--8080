✨ Features

· Direct public URL – without any advertisement / warning pages
· Coloured terminal output for better readability
· Port validation – only accepts valid port numbers (1–65535)
· Auto-detection of ssh command; warns if missing
· Cross-platform: works on Termux (Android), Linux, macOS, Windows

---

📦 Requirements

· Python 3
· openssh (SSH client) – pre-installed on most systems; if not:
  ```bash
  # Termux
  pkg install openssh
  # Debian/Ubuntu
  sudo apt install openssh-client
  ```
